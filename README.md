# SafeCombat - MC 1.20.1

Avoiding anti-gambling in PvP

## Commands

- **/protection**
    - **disable**  
    Disable your protection
    - **remove**  
    Unprotect a player
    - **add**  
    Protect a player

## Permissions

- **safecombat.protection**  
    Add and remove player's protection

# Dependencies

- **WorldEdit** for **WorldGuard**
- **SaberFactions**
